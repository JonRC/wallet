## Get Started
### Requirements: Docker, Docker Compose

**Run migrations**
```bash
./run-migration.sh
```

**Run project**
```bash
docker compose up --build
# or
docker compose up --build --watch
```

**Endpoints**  
API: http://localhost:4000  
Documentation: http://localhost:4000/docs
